# Developmental Trauma Disorder (DTD) vs PTSD / Complex PTSD (C-PTSD)

> **Note:**  
> - **DTD** is a *proposed* diagnosis (primarily associated with Bessel van der Kolk and colleagues) and is **not officially included** in DSM-5 or ICD-11.  
> - **PTSD** is officially recognized in DSM-5 and ICD-11.  
> - **Complex PTSD (C-PTSD)** is officially recognized in **ICD-11**, not DSM-5.

| **Developmental Trauma Disorder (DTD)** | **PTSD + C-PTSD Criteria** |
|---|---|
| ## A. Developmental Trauma Exposure  | ## PTSD Core Criteria |
| **Both required:** | Typically includes exposure to actual or threatened death, serious injury, or sexual violence. |
| - Interpersonal victimization (physical abuse, sexual abuse, violence, etc.) | ### PTSD symptom clusters (ICD-11 simplified model) |
| - Disruption of attachment with primary caregiver (neglect, abandonment, emotional abuse, prolonged separation) | - Re-experiencing the trauma in the present (flashbacks, nightmares, intrusive memories) |
|  | - Avoidance of trauma reminders |
|  | - Persistent sense of current threat (hypervigilance, exaggerated startle) |
| ## B. Emotion / Somatic Dysregulation |  |
| At least several of the following: | ## Additional C-PTSD Criteria ("Disturbances in Self-Organization") |
| - Extreme emotional reactivity or shutdown | ### 1. Affect dysregulation |
| - Somatic distress without medical explanation | - Difficulty calming down |
| - Reduced awareness of emotions/body states | - Emotional numbness or explosive emotions |
| - Difficulty describing feelings (alexithymia) |  |
|  | ### 2. Negative self-concept |
| ## C. Attentional / Behavioral Dysregulation | - Persistent feelings of worthlessness, shame, guilt, or failure |
| At least several of the following: | - Feeling damaged, defeated, or fundamentally defective |
| - Hyperfocus on threat or inability to recognize danger |  |
| - Poor self-protection / risk-taking | ### 3. Relational disturbances |
| - Maladaptive self-soothing | - Difficulty sustaining relationships |
| - Self-harm | - Feeling distant or cut off from others |
| - Difficulty sustaining goal-directed behavior | - Persistent distrust or inability to feel close to people |
|  |  |
| ## D. Self / Relational Dysregulation | ## Functional Impairment |
| At least several of the following: | Symptoms cause clinically significant distress or impairment in functioning. |
| - Persistent negative self-view |  |
| - Attachment insecurity/disorganization |  |
| - Distrust and betrayal expectations |  |
| - Reactive aggression |  |
| - Boundary problems |  |
| - Impaired empathy or over-identification with others' distress |  |
|  |  |
| ## E. PTSD-like Symptoms |  |
| At least some symptoms from PTSD clusters (intrusions, avoidance, arousal) | PTSD symptoms are required for both PTSD and C-PTSD. |
|  |  |
| ## F. Duration / Impairment |  |
| Symptoms persist over time and impair functioning across developmental domains. | Symptoms persist for weeks/months and impair social, occupational, or personal functioning. |

## Key Difference Summary

| **DTD** | **C-PTSD** |
|---|---|
| Focuses specifically on **developmental and attachment trauma in childhood** | Focuses on the effects of **prolonged or repeated trauma** |
| Emphasizes developmental disruption across emotional, behavioral, relational, and somatic systems | Defined as **PTSD + disturbances in self-organization (DSO)** |
| Proposed primarily for children/adolescents | ICD-11 diagnosis often applied in adults as well |
| Includes attachment disruption as central | Attachment issues are common but not required as causal criteria |
